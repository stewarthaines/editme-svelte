# 24. Cover creator

## Brainstorming

In the `metadata` view provide a simple ui for the user to create a cover image. Use SVG for the source, and generate a fallback PNG item as required. The user should be able to choose a colour scheme, a simple geometric cover background (gradients and shapes) and enter up to three text items with style and position. The cover is a simple SVG object. This process creates a new manifest item for the cover and another for PNG fallback, and stores parameters for the cover generation in the workspace `SOURCE/settings.json` so it can be modified as for other content.
