# 30 Package EPUB

## Brainstorming

There is a button in the top row of the sidebar (next to `EDITME.html` label when sidebar is expanded, next to the disclose arrow when the sidebar is collapsed). When 'advanced mode' is true, and the sidebar is expanded it is an action button 'Package' that causes the current workspace to be packaged and a file download started.

When the sidebar is collapsed, the button sits below the disclose arrow and just has a download icon (no label).

In the 'workspace' view, where all the workspaces are listed, each workspace has a button to export a 'dead' version of the epub which doesn't contain the EDITME.html, SOURCE.zip or about_editme.xhtml manifest items. requires re-writing the workspace content.opf to be consistent with the reduced manifest item list.
